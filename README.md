# 抖音评论采集器 (Douyin Comment Scraper)

## 项目简介
本项目是一个基于 Python 和 Playwright 的自动化工具，旨在采集抖音网页版上特定话题（如“复刻台球杆”）的视频评论。主要用于市场分析，筛选出具有购买意向（如“怎么买”、“价格”）的用户评论。

## 功能特性
*   **自动化搜索**: 自动在抖音搜索指定关键词。
*   **时间筛选**: 自动筛选发布时间（如“过去一周”），并进一步精确过滤出最近3天内的视频。
*   **评论提取**: 采集视频下的顶级评论。
*   **意向识别**: 根据预设关键词（“怎么买”、“价格”等）筛选高价值评论。
*   **数据导出**: 将结果保存为 CSV 文件。
*   **抗干扰**: 模拟真实用户行为（随机延迟、滚动）以降低被检测风险。

## 快速开始

### 前置要求
*   Python 3.10+
*   Chrome / Chromium 浏览器

### 安装
1.  克隆或下载本项目。
2.  安装依赖:
    ```bash
    pip install -r requirements.txt
    playwright install chromium
    ```

### 运行
```bash
python src/main.py
```
